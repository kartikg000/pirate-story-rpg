# Master Prompt (Future Reference)

Build a Roblox 3D pirate simulation story game in Luau with secure server-authoritative architecture.

## Goals
- PG-13, moderation-safe content
- 30-level progression achievable in ~20 days
- Story starts with first-person chicken-steal chase, then island field reunion, harbor departure on old pirate ship
- Coins unlock character skins/clothes over progression
- Core loops: sailing, hunger, hunting sea monsters, quests
- World simulation: storms, tsunamis, calm/good days, snow events, summer events, beautiful sunsets/sunrises, pink and blue skies, clouds, moon phases, stylized water, distant trees

## Technical Constraints
- Roblox Studio + VS Code + Rojo
- Server-authoritative economy/progression/combat
- Validate all remotes (type/range/cooldowns/state)
- DataStore retries + session lock
- Clear module boundaries: Shared / Server / Client

## Delivery style per milestone
1. Files created/updated
2. How to test in Studio
3. Security checks added
4. `docs/progress.md` updated

## Story Prize
Final treasure: **Heart of Tides**.
Reward includes unique ocean control ability + world ending choice.
