# Pirate Story RPG (Roblox)

A 3D pirate simulation story game built with Roblox Studio + Luau + VS Code (Rojo workflow).

## Stack
- Roblox Studio
- Luau
- VS Code
- Rojo
- Git/GitHub

## Quick start
1. Install Roblox Studio.
2. Install Rojo: https://rojo.space/
3. Open this folder in VS Code.
4. Start Rojo server:
   - `rojo serve`
5. In Roblox Studio, install Rojo plugin and connect to `localhost:34872`.
6. Press Play in Studio.
